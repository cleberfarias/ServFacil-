# ServiFácil

Aplicativo React Native para contratação de serviços entre clientes e prestadores.

## Estrutura Inicial
- Splash/Boas-vindas
- Login/Cadastro
- Home (Cliente e Prestador)
- Perfil do Prestador
- Contratação/Pagamento
- Chat
- Avaliação

## Tecnologias sugeridas
- React Native
- React Navigation
- (Preparado para integração futura com backend Node.js/Firebase)

## Como iniciar
1. Instale as dependências:
   ```
   npm install
   ```
2. Rode o app:
   ```
   npx expo start
   ```

## Estrutura de pastas
- `/src/screens` — Telas do app
- `/src/components` — Componentes reutilizáveis
- `/src/navigation` — Configuração de navegação

---

Este projeto é um MVP e pode ser expandido conforme necessidade.
